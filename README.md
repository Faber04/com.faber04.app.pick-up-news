### v1.3.0
- 🔍 Auto-detect JSON feeds from websites (checking common locations like feed.json, /api/feed, .well-known/feed.json)

- 📋 Support for JSON Feed format (https://jsonfeed.org/)
- 🌐 Auto-discovery via HTML <link> tags (rel="alternate" type="application/feed+json")
